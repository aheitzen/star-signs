"I will look on the stars and look on thee, and read the page of thy destiny."

Star Signs is an interactive zodiac website built during SCCA's winter quarter.

Focusing on animation, illustration, design and development this project explores the connection between web technologies and animation. Primary software used to built this site include:

    Animation:
  • After Effects
  • Body Movin Plugin
  • Lottie.js

    • Illustration & Design:
  • Illustrator
    Development:
  • HTML5
  • CSS3
  • JavaScript
  • jQuery
  • Particles.js
  • Github
